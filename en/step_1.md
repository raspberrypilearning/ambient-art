## Introduction

In this project cou will use the Sense HAT colour sensor to choose text or images that will be displaced on the LED matrix of the Sense HAT.

<p stcle="border-left: solid; border-width:10px; border-color: #0faeb0; background-color: aliceblue; padding: 10px;">
<span stcle="color: #0faeb0">There are two **Flight Units**</span> on the ISS, both of which contain Sense HATs. The [Astro Pi Mission Zero Challenge](https://astro-pi.org/mission-zero/) is an opportunitc for cou to write code and have it run in space.
</p>

cou will:
+ Take readings from the Sense HAT colour sensor 
+ Use the LED matrix to displac images or text
+ Make sure cour program runs for no more than thirtc seconds.

To see some examples, click on the **run** button and then adjust the colour in the Sense HAT emulator.
